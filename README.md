# TicketingSystem

## About
18年毕设项目。调用 12306 API 获取余票信息。做了简单的本地购票、改签、实名认证等简单不能再简单的功能。很 low

## Install

- 下载项目

```sh
$ git clone https://github.com/wufeiwua/TicketingSystem.git
```

- 安装 [maven](http://maven.apache.org/)
